录音基础概念

发表于2025-11-14|更新于2025-11-14|基础
|阅读量:

声音的基础知识

一个声音包含物理特性和声音的感知特性

  • 物理特性：
特性 作用
频率 决定音调的高低，单位是赫兹 (Hz)。
振幅 决定音量的大小，通常用分贝 (dB) 来衡量。
波形 声音的形状，不同乐器或声音的波形不同。
  • 声音的感知: 响度，音色，掩蔽效应

人耳能听到的频率范围通常是 20 Hz~20 kHz。
掩蔽效应是指一个声音的存在会使另一个声音难以听到。

录音参数

  • 采样率 (Sample Rate)：
    每秒钟对音频信号进行采样的次数，单位是赫兹 (Hz)。 采样率越高，音频信号的精度越高。

  • 位深 (Bit Depth)：
    每个音频样本的位数，决定了音频信号的动态范围和精度。

  • 码率 (Bit Rate)：
    单位时间内传输或存储的音频数据量，通常以千比特每秒 (kbps) 为单位。 码率越高，音频文件包含的数据量就越大，音质也越好。

  • 声道 (Channels)：
    单声道 (Mono)、立体声 (Stereo)、多声道 (如 5.1 声道、7.1 声道)。

常见采样率： 44.1 kHz (CD 质量)、48 kHz (视频制作)、96 kHz 或 192 kHz (专业录音)。
常见位深： 16 位 (CD 质量)、24 位 (专业录音)。

录音文件大小计算

  • 在无损无压缩格式当中：
    码率 = 采样率 * 位深度 * 声道数
    所以文件大小 (字节) = 码率 * 录音时长 (秒) / 8 = 采样率 * 位深度 * 声道数 * 录音时长 (秒) / 8

  • 在有损压缩格式当中：
    码率便不等于上面的公式了，因为原始信息已经被破坏，文件大小的计算就通过码率来计算
    文件大小 (字节) = 码率 * 录音时长 (秒) / 8

无损压缩和有损压缩

  • 无损压缩指的是在无损格式之间的压缩(转换)，无论压缩(转换)成什么格式，音质都是相同的，并且都能还原成最初同样的文件。
  • 有损压缩指的是声音信息在压缩过程中发生了丢失，且所丢失的声音无法用采样率和位数表示出来，但特点就是压缩后的文件变的很小。

常见的无损格式：wav, flac, ape
常见的有损格式：mp3, aac, wma
